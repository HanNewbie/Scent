#### Install Project
```bash
    git clone https://github.com/HanNewbie/Scent.git
```
```bash
    composer install
```
-  copy env file lalu sesuaikan dengan database
```bash
    php artisan migrate:fresh --seed
```
```bash
    php artisan storage:link
```
- Runing program
```bash
   php artisan serve
```
